## This section is dedicated to creating custom moons for Lethal Company.

### This repository contains two approaches to custom moon creation:

- Using **`DawnLib`**: [Moons/DawnLib](https://github.com/xw1w1/LC-Modding-Guidelines/tree/main/Lethal%20Company/DawnLib)
- Using **`Lethal Level Loader`**: [Moons/LethalLevelLoader](https://github.com/xw1w1/LC-Modding-Guidelines/tree/main/Lethal%20Company/LethalLevelLoader)

**You need to start from the [editor setup guide](https://github.com/xw1w1/LC-Modding-Guidelines/blob/main/Lethal%20Company/1.%20Setup.md) anyway.**

Then you should use references named `1. Project Setup`, then `2. Mod Setup`, then `Moons/1. Moon Setup` in folder of your selected modding API.
