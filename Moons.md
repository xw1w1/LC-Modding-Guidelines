## This section is dedicated to creating custom moons for Lethal Company.

### This repository contains two approaches to custom moon creation:

- Using **`DawnLib`**: https://github.com/xw1w1/LC-Modding-Guidelines/tree/main/en/DawnLib
- Using **`Lethal Level Loader`**: https://github.com/xw1w1/LC-Modding-Guidelines/tree/main/en/LethalLevelLoader

**But any way, you need to follow the [editor setup guide](https://github.com/xw1w1/LC-Modding-Guidelines/blob/main/en/1.%20Setup.md) anyway.**

Then you should use references named `1. Project Setup`, then `2. Mod Setup`, then `Moons/1. Moon Setup` in folder of your selected modding API.
